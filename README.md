<h2 align="justify">
:v: Descripcion del proyecto :wink:
</h2>
<p align="justify"> 
Se trata sobre la obtencion de informacion de una base de datos, el manejo de la informacion y graficacion de los datos.
</p> 


<h3 align="justify">Funcionamiento</h3>

<p align="justify">
  
:heavy_check_mark: `Funcionamiento 1:` Se obtuvo la informacion de los datos necesarios de 4 tiendas a traves de url.

:heavy_check_mark: `Funcionamineto 2:` Una vez obteniendo la informacion se procedio a atender las demandas del proyecto y realizar 
operaciones para tener promedios y totales de los productos solicitados.

:heavy_check_mark: `Funcionamiento 3:` Una vez teniendo los resultados se graficaron para poder tener un mejor entendimiento de la situacion de las tiendas.

:heavy_check_mark: `Funcionamiento 4:` Al final se relizo un analisis para poder dar una recomendacion informada de cual tienda debia ser vendida.
</p>
